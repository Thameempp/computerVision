# Computer Vision

OpenCV examples for reading images and videos, webcam capture, resizing and rescaling, drawing shapes and labels, and image transformation.

## Project Structure

- `reading_images_videos/` - image, video, and live webcam examples
- `resizing_rescaling/` - resize and rescale examples for images, videos, and webcam frames
- `drawing_shapes+labels/` - drawing shapes and adding labels with OpenCV
- `image_transformation/` - image transformation examples
- `captured_frames/` - saved frames from webcam/video runs
- `resources/` - supporting files and sample assets
